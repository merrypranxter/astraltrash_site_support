README / mutation instructions

Where the editable stuff lives

PALETTE contains every reusable color. PARTS contains the sprite as intentional rectangles in the format [colorName, x, y, width, height]. Change those integers and the pixels change. The 64×128 canvas stays transparent.

How this becomes a sprite sheet

Add new part variants, then add frames to animations. Each frame can offset body groups and swap overlays. A future exporter can draw each frame into adjacent 64×128 cells, producing a standard horizontal PNG sprite sheet.
